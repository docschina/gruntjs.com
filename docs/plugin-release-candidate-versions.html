Plugin Release Candidate Versions

Note: because these are pre-release versions, please follow the instructions in the "Installing a published development version of Grunt" section of the Installing Grunt guide. And as a reminder, do not specify a version range (using ~ or <=) when including a pre-release version of Grunt or Grunt plugin in your project.
